# xCVM

xCVM - Character Virtualization Model is a standard library for modelization 3D human for Animation, Digital graphics, and so on.

Its standard is the implementation on Unity3D, based on [UniVRM](https://github.com/vrm-c/UniVRM) and fully support [glTF 2.0](https://www.khronos.org/gltf/)

## Features

xCVM supports the [VRM standards, including VRM `0.x` and `1.x`](https://github.com/vrm-c/vrm-specification), as well as the [glTF 2.0 specification](https://registry.khronos.org/glTF/).

Also, xCVM supports the extended `.xcvm` format for better compression.

## Supported Environments

The latest xCVM supports Unity 2021.3 LTS or later with backends both .NET and IL2CPP.
